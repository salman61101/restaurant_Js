# Restaurant Page

A modern restaurant homepage built with **HTML**, **CSS**, **JavaScript**, **Webpack**, and **ES6 Modules**.

---

## 📖 Overview

This project demonstrates how to build a multi-page website using JavaScript only. Instead of creating multiple HTML pages, all page content is dynamically rendered into the DOM using ES6 modules and event-driven navigation.

The project also introduces Webpack for module bundling and modern JavaScript project organization.

---

## ✨ Features

- Dynamic page rendering
- Home, Menu, and Contact pages
- Navigation without page reloads
- Modular JavaScript architecture
- Webpack development environment
- Responsive layout
- Modern UI

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/salman61101/restaurant_Js.git
```

Navigate into the project:

```bash
cd restaurant_Js
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Build the production version:

```bash
npm run build
```
